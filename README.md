# Sales Data Analyzer (Python)
Ein einfaches Python-Skript zur automatisierten Datenbereinigung und Visualisierung von Verkaufsdaten. 

**Technologien:**
* Python 3
* Pandas (Datenverarbeitung & CSV-Handling)
* Matplotlib (Datenvisualisierung)

**Funktionsweise:**
Das Skript generiert fiktive IT-Sales-Daten, speichert diese als `.csv`-Datei, liest sie mit Pandas wieder ein, gruppiert die Umsätze nach Produktkategorien und erstellt abschließend ein Balkendiagramm zur visuellen Analyse. Dieses Projekt zeigt die Grundlagen der Datenverarbeitung (ETL-Prozess) mit Jupyter Notebooks / Google Colab.
